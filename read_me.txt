

**StalkTrade Pro** is a Python-based desktop application that simulates a stock market environment. It allows users to **track live stock prices**, **view interactive charts**, **simulate trades**, and **manage a personal portfolio and watchlist** — all without using real money.

This project is great for **beginners who want to learn stock trading** in a safe, offline environment using real data from the internet.

---



- 🔐 Secure login and registration system
- 📊 Real-time stock price tracking using the `yfinance` API
- 📈 Line and candlestick chart views using `matplotlib`
- 💼 Simulated buy/sell trading with portfolio updates
- 📋 Watchlist and portfolio management
- 📰 Market news tab (simulated headlines)
- 🌙 Modern dark-themed user interface
- 🔄 Auto-updating prices every few seconds

---



- **Python**  
- **Tkinter** – GUI development  
- **yfinance** – API for stock market data  
- **Matplotlib** – For drawing charts  
- **JSON** – User data storage  
- **hashlib** – For password hashing  
- **Requests + BeautifulSoup** – (for optional news scraping)
